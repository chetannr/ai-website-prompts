### 3. Typography Restraint

**Current**: "Maximum 2 fonts"  
**Their Feedback**: "Why not one? Or system fonts?"

**Refinement**:
- Consider single font families
- System font stack as default
- When to add a second font (sparingly)
- Typography hierarchy over font variety

**Example Addition**:
```markdown
### Typography Philosophy

**Start with System Fonts**
- Fast, familiar, accessible
- No loading time
- Native feel

**Add One Font Only If:**
- Brand requires distinctive typography
- Content benefits from specific character
- Performance impact is minimal

**Never Add More Than:**
- One heading font
- One body font
- Total: 2 fonts maximum (prefer 1)
```

---

### 4. Color Restraint

**Current**: "Maximum 3-4 colors"  
**Their Feedback**: "Why not monochrome? Or 2 colors?"

**Refinement**:
- Start with monochrome
- Add color only when necessary
- Consider single accent color
- Black, white, one color

**Example Addition**:
```markdown
### Color Philosophy

**Start Monochrome**
- Black, white, grays
- Maximum clarity
- No distraction

**Add One Color When:**
- Brand requires it
- Functional purpose (CTAs, states)
- Emotional connection needed

**Maximum Palette:**
- Base: Black, white, 2-3 grays
- Accent: 1 color (if needed)
- Total: 5-6 colors maximum
```

---

### 5. Motion Restraint

**Current**: "Subtle animations"  
**Their Feedback**: "Why animate at all? What's the purpose?"

**Refinement**:
- Default: No animation
- Add only for functional purpose
- Micro-interactions for feedback
- Never decorative motion

**Example Addition**:
```markdown
### Motion Philosophy

**Default: No Animation**
- Faster, simpler, clearer
- No distraction
- Immediate feedback

**Add Motion Only For:**
- Functional feedback (button press)
- State changes (modal open/close)
- Loading states (if necessary)
- Never for decoration

**If You Must Animate:**
- 200-300ms maximum
- Ease-in-out only
- Subtle, not noticeable
```
